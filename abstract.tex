In this study, a hybrid cryptographic system has been designed and implemented to provide confidentiality, integrity, authentication and non-repudiation for secure email communication. The system is based on an Alice (sender) to Bob (receiver) communication scenario and is supported by a desktop application interface that visualises the cryptographic operations of both parties step by step.

In the proposed system, the SHA-256 hash function and RSA-2048 PSS (Probabilistic Signature Scheme) digital signature algorithm are used for message integrity and authentication. Message confidentiality is provided by the symmetric encryption algorithm AES-256-GCM (Galois/Counter Mode); the symmetric session key is asymmetrically encrypted using RSA-2048 OAEP padding. This hybrid approach combines the security advantages of asymmetric cryptography with the performance advantages of symmetric cryptography.

The PyQt6-based desktop application presents the inner workings of the AES-256, RSA-2048 and SHA-256 algorithms in a pedagogically correct manner through animations of the 14-round AES state matrix, Extended Euclidean Algorithm steps and SHA-256 compression circuit. The application has been validated with 26 unit tests and uses cryptographic primitives compliant with NIST FIPS 180-4, NIST SP 800-38D and RFC 8017 standards.
